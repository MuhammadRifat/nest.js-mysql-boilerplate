# Boilerplate Nest.js and MySQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run prod
```

## CRUD Generate

```bash
$ npm run generate:crud module-name
```

## DB Migration

```bash
$ npm run migration
$ npm run migration:all
$ npm run migration:rollback
$ npm run seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
